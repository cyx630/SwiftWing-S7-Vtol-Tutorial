VtolS7飞控使用教程

无线连接VtolS7飞控

  • 电脑连接SIYI通信链路的串口（部分电脑连接通信链路后可以自动连接上飞控，无需配置）
  • 根据电脑端口，设置qgc连接
  • 如下配置qgc连接，点击确认即可

校准VtolS7飞控

  • 需要校准的情况：全新飞控刷写固件参数后、硬件变动、飞行状态异常、长时间未使用。

罗盘校准

  • 保证无人机静止，水平放置。点击QGC左上角图标菜单 -> Vehicle Setup

  • 点击，传感器

  • 点击，罗盘
  • 点击，OK

  1. 按照提示依次摆放对应姿态后逆时针旋转无人机

陀螺仪校准

  • 保证无人机静止，水平放置。点击QGC左上角图标菜单 -> Vehicle Setup

  • 点击，传感器

  • 点击，陀螺仪
  • 点击，OK

加速度计

  • 保证无人机静止，水平放置。点击QGC左上角图标菜单 -> Vehicle Setup

  • 点击，传感器

  • 点击，加速度计
  • 点击，OK

  1. 按照提示，依次摆放无人机姿态后静止

校平地平线

  • 保证无人机静止，水平放置。点击QGC左上角图标菜单 -> Vehicle Setup

  • 点击，传感器

  • 点击，校平地平线
  • 点击，OK

  • 成功校平地平线后如下

空速校准

  • 保证无人机静止，水平放置。点击QGC左上角图标菜单 -> Vehicle Setup

  • 用手挡住空速计，不要让环境风吹到空速

  • 点击，传感器

  • 点击，空速
  • 点击，OK

  • 提示下面信息时，用力朝空速计吹气
