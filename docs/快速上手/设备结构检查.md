# 设备结构检查

## 飞行前检查

* 注意检查倾转电机座是否存在开裂，避免飞行中出现事故

  ![1767171348045](image/设备结构检查/1767171348045.png)
* 注意尾撑是否存在裂纹，若存在裂纹，建议及时修复

![1767171294314](image/设备结构检查/1767171294314.png)

* 注意平尾与机翼是否平行，若未调平，需调平后飞行，避免影响飞行性能

![1767171303770](image/设备结构检查/1767171303770.png)

* 注意翼面是否存在开裂，若存在开裂，建议及时修复，避免影响飞行性能

![1767172080661](image/设备结构检查/1767172080661.png)

<img src="image/设备结构检查/1767171319146.png" alt="1767171319146" style="transform: rotate(-90deg);">

* 注意机身是否存在裂纹，若存在裂纹，建议及时修复，避免影响飞行性能

![1767172103718](image/设备结构检查/1767172103718.png)
